Core Web Vitals

Ruhiger Main-Thread

Die grosse bewegte Flaeche liegt in einem Fragment-Shader, nicht in vielen einzelnen DOM-Updates. Das Layout bleibt still, die Animation laeuft auf der GPU.

Namen sind der Vertrag.

Contract Check

Ein Buchstabe genuegt

Heisst es im JavaScript uColor und im GLSL uColour, gibt es keine Fehlermeldung. Es gibt nur ein falsches Bild. Der Vertrag wird nicht erzwungen.

varying

Drei Ecken, eine Flaeche

Der Vertex-Shader setzt vUv an den Ecken, die GPU interpoliert ueber die Flaeche, der Fragment-Shader liest pro Pixel. So wird aus drei Werten ein weicher Verlauf.

uniform

Ein Wert pro Frame

uTime ist eine einzige Zahl, die die CPU pro Bild setzt. Millionen Pixel lesen sie gleichzeitig und bewegen die Flaeche. Die teure Arbeit bleibt auf der GPU.
